Портфолио

Занимаюсь программированием как хобби с раннего детства, а с 2019 еще и веб-разработкой. Помимо небольших традиционных веб-приложений и страниц также имею опыт разработки интерактивной рекламы, портативных приложений и написания скриптов. Далее следуют примеры работ за последние несколько лет с кратким оисанием:

Gatos

TS React Vite Vercel Supabase

Каталог мемов про котов из одноименной группы ВК. Идея проекта несколько раз использовалась для отработки навыков разработки с применением веб-фреймворков и сервисов. Последняя версия использует Postgres в качестве базы данных с использованием сервисов Supabase, полноценую серверную часть с авторизацией и пагинацией запросов к базе данных. Фронтенд построен с использованием React.

GitHub Demo

Онлайн-реализация игры Морской бой с поддержкой нескольких сессий. Использует Node.js с применением WebSocket для серверной части. Клиентская часть использует эзотерический подход с применением пользовательских шрифтов для отрисовки интерфейса.
