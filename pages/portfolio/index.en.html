Portfolio

Doing programming as a hobby and wishing to become an indie game developer since I was young, I was also doing web stuff since 2019. This includes not only small-scale fullstack and frontend projects, but also playable ads, standalone apps and scripts. Here is a brief overview of my accomplishments during last few years.

Gatos

TS React Vite Vercel Supabase

Gatos is a directory of tagged memes about cats built with React. The idea of the project is used to learn about web frameworks in practice. Latest React version features Supabase DB, full-fledged backend with custom authorization and database paging and frontend.

GitHub Demo

Previous version, built with Quazar (Vue) and Firebase

Arstotzka

JS

Arstotzka is NPM package for JS data validation, made primarily for validating API responses. Unlike alternatives, it offers laconic schema.

Battleships

JS Node.js WebSocket

Multi-session online game of Battleships, using Node.js server for managing WebSocket connections between players. Client features esoteric UI built with custom font-faces.

GitHub Demo

Strelitzia

JS Electron Appstore API

Strelitiza is standalone Electron app that provides convenient user interface for Apple's AppStore services, using their undocumented backend. Abandoned due to lack of access to developer account, currently unusable due to API changes. Uses Electron and Electron Forge, Handsontable, inspired by Fastlane's Spaceship
